Terminology

agent

The direct performer or driver of the action; the reviewer.

object

The object upon which the action is carried out, whose state is kept intact or changed, i.e. the item which is under review.

result

The result produced in the action, i.e. the result of the review.

startTime

The date or time that the review began.

endTime

The date or time that the review concluded.
